# Anomaly Live Detection

## Overview
Anomaly Live Detection is an AI-powered video surveillance system that detects abnormal crowd behavior in real time using computer vision and deep learning techniques. The system processes live video streams, identifies objects, and detects unusual activities to enhance security and monitoring.

## Features
- Real-time video analysis
- Crowd behavior monitoring
- Anomaly detection using deep learning
- Object detection with YOLOv8
- Automated surveillance for security applications
- Real-time visualization of detection results

## Technologies Used
- Python
- OpenCV
- TensorFlow
- Keras
- YOLOv8
- NumPy
- Pandas
- SQL Server

## Project Workflow
1. Capture live video stream.
2. Preprocess video frames.
3. Detect objects using YOLOv8.
4. Analyze crowd behavior using deep learning.
5. Identify abnormal activities.
6. Display real-time detection results.

## Installation
1. Clone the repository:
```bash
git clone https://github.com/your-username/anomaly-live-detection.git
```

2. Navigate to the project folder:
```bash
cd anomaly-live-detection
```

3. Install the required dependencies:
```bash
pip install -r requirements.txt
```

4. Run the application:
```bash
python main.py
```
